Sebuah Website penjualan produk secara online, untuk mempermudah pelanggan dalam melakukan pembelian. program tersebut sudah terintegrasi dengan API Raja Ongkir untuk memastikan biaya pengiriman barang sesuai jarak pengiriman dari toko ke pelanggan. terdapat fitur transaksi pembayaran, penerimaan, riview, chating, kategori per produk, detail produk, diskon produk dan produk unggulan. bagi pihak pengelola terdapat fitur Pengelolaan kategori produk, data produk, data diskon produk, transaksi penjualan, laporan produk penjualan dan stock.
